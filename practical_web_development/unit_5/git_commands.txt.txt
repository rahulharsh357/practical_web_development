Git Commands for Web Technology Practical

1. Initialize Git repository
git init

2. Check status of files
git status

3. Add all files to staging area
git add .

4. Commit changes
git commit -m "Web Technology Practical Submission"

5. Rename branch to main
git branch -M main

6. Add remote GitHub repository
git remote add origin https://github.com/rahulharsh357/practical_web_development.git
7. Push files to GitHub
git push -u origin main
